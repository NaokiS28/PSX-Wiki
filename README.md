## About

PSX-Wiki is a technical reference focused on the internals of the Sony PlayStation. It is intended for developers, researchers, and enthusiasts working on emulators, homebrew, tools, or preservation projects.

This is **not** a general gaming wiki. Content is scoped to technical documentation, reverse-engineered specifications, and development resources.

## SDKs & Development Tools

PSX-Wiki covers both official Sony SDKs and community-developed open-source alternatives, including toolchain setup, library references, and usage notes relevant to PS1 development and homebrew.

**Recommended**
- [**PSYQo**](https://github.com/grumpycoders/pcsx-redux/tree/main/src/mips/psyqo) — Modern C++17 framework for PS1 homebrew; part of the PCSX-Redux project
- [**ps1-bare-metal**](https://github.com/brad-lin/ps1-bare-metal) — Bare-metal PS1 development without an SDK; useful for low-level learning and research

**Open Source / Homebrew**
- [**PSn00bSDK**](https://github.com/Lameguy64/PSn00bSDK) — Modern open-source C/C++ SDK for PS1 homebrew
- [**PSXSDK**](https://github.com/nathanhi/psxsdk) — Lightweight alternative open-source SDK

**Official**
- **PsyQ** — Primary official SDK; includes linker, make tools, and a full C runtime
- **SDevTC** — Sony's official compiler toolchain based on Cygnus GCC
- **DTL-H Series** — Official hardware devkits (DTL-H2000, DTL-H2500, DTL-H505, etc.)

Contributions are welcome. Before submitting:

- Verify claims against hardware tests, SDK documentation, or trusted sources
- Cite sources where possible (unverified information should be clearly marked)
- Follow the existing page structure and formatting conventions
- Open an issue before making large structural changes
- **Do not submit or post copyrighted material without explicit permission from the original rights holder.** This includes but is not limited to official Sony documentation, licensed SDK materials, and proprietary tools. When in doubt, do not post it.
